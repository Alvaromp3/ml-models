PASOS PARA CREAR APLICACIÓN FLASK - PREDICTOR DE VINOS
=====================================================

ORDEN DE CREACIÓN DE ARCHIVOS:

1. MODELO.PY (PRIMERO - BASE DEL PROYECTO)
   - Contiene toda la lógica de machine learning
   - Funciones: limpiar_datos(), entrenar_modelo(), predecir()
   - Debe crearse primero porque app.py lo importa

2. REQUIREMENTS.TXT (SEGUNDO - DEPENDENCIAS)
   - Lista todas las librerías necesarias
   - Flask, pandas, numpy, scikit-learn
   - Permite instalar dependencias con: pip install -r requirements.txt

3. CREAR CARPETA TEMPLATES/ (TERCERO - ESTRUCTURA)
   - mkdir templates
   - Flask busca archivos HTML aquí automáticamente

4. TEMPLATES/INDEX.HTML (CUARTO - INTERFAZ)
   - Interfaz web del usuario
   - Formulario para introducir datos del vino
   - JavaScript para comunicarse con Flask

5. APP.PY (QUINTO - SERVIDOR WEB)
   - Aplicación Flask principal
   - Rutas: /, /entrenar, /predecir
   - Conecta modelo.py con templates/index.html

6. README.MD (SEXTO - DOCUMENTACIÓN)
   - Instrucciones de instalación y uso
   - Descripción del proyecto

COMANDOS PARA EJECUTAR:
======================
1. cd vinos_modelo
2. pip install -r requirements.txt
3. python3 app.py
4. Abrir: http://localhost:5000

FLUJO DE LA APLICACIÓN:
======================
Usuario → index.html → app.py → modelo.py → respuesta → index.html → Usuario
